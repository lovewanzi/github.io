## 1、python的传参

今天写力扣时，发现一个有意思的传参，代码大概如下。

```python
a = []
def aa(b):
    if len(b) == 3:
        return
    aa(b.append(1))
aa(a)

print(a)

#执行结果
"C:\Users\xiaow\Desktop\alg_practice\main.py", line 5, in aa
    aa(b.append(1))
TypeError: object of type 'NoneType' has no len()
```

这段代码中`b.append(1)`是没有返回值的，所以传给下一层的是一个None值。因此在调用下一层递归时，会因为`None.append(1)`报错
